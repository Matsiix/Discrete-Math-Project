Linear Search — Proof

Show / Hide

Preserved predicate: At the beginning of iteration i, the target does not appear in indices 0…i−1.

Initialization: before the loop i = 0 ⇒ predicate vacuously true.

Maintenance: if arr[i] ≠ target and predicate true, incrementing i preserves it.

Termination: loop ends when found or i = n ⇒ by predicate value not present.

Worst-case complexity: O(n). Each iteration is O(1), max n iterations.

Binary Search — Proof

Show / Hide

Preserved predicate: The target (if present) lies in [L,R].

Inductive proof: for interval length k; splitting reduces k → induction applies.

Recurrence: \( T(n) = 2T(n/2) + O(n) \rightarrow O(n \log n) \) .

Precondition: array must be sorted.

Bubble Sort — Proof

Show / Hide

Preserved predicate: after i passes, last i elements are in final position.

Counting comparisons: \( \sum_{i=1}^{n-1} (n - i) = \frac{n(n-1)}{2} \) .

Correctness idea: each pass bubbles largest remaining element to end; repeat until sorted.

Merge Sort — Proof

Show / Hide

Recursive invariant: mergeSort(A) returns a sorted version of A.

Strong induction: base n≤1 trivial; combine two sorted halves via merge.

Recurrence: \( T(n) = 2T(n/2) + O(n) \rightarrow O(n \log n) \) .
